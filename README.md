## 🧠 System Overview

<table>
<tr>
<td width="60%">

CareerSync is an **AI-powered resume screening engine** designed to eliminate manual hiring bottlenecks.

It intelligently analyzes resumes, extracts meaningful insights, and matches candidates with job descriptions using advanced **NLP + ML techniques**.

> ⚡ Think of it as your automated recruiter in code form.

</td>
<td width="40%" align="center">

```
  ┌─────────────────────┐
  │   CareerSync Core   │
  │  ─────────────────  │
  │  ✦ Parse Resumes    │
  │  ✦ Extract Skills   │
  │  ✦ NLP Matching     │
  │  ✦ Smart Ranking    │
  └─────────────────────┘
```

</td>
</tr>
</table>

## 🚀 Core Features

<div align="center">

| 🔍 Feature | ⚡ Description |
|:---:|:---|
| 📄 **Resume Intelligence** | Parses resumes and extracts key structured data |
| 🧠 **NLP Understanding** | Semantic analysis of candidate profiles vs job roles |
| 🎯 **Smart Matching** | Matches candidates to roles using cosine similarity |
| 📊 **Ranked Output** | Profiles ranked by relevance score, instant shortlist |
| ⚡ **Speed** | Eliminates manual review — decisions in seconds |

</div>
